# 🤖 ИИ-бот Фонда Шахмардана Есенова

Телеграм-бот, который отвечает на вопросы о грантах, стипендиях и программах
Фонда Есенова. Работает на **RAG** (поиск по базе знаний) + **Gemma** (LLM).
Отвечает на русском, казахском и английском — на языке вопроса.

```
Вопрос → [RAG: поиск фрагментов в базе] → [Gemma генерирует ответ] → ответ + источники
```

## 🚀 Запуск за 5 шагов

### 1. Установить зависимости
```bash
cd day_5
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Настроить ключи
```bash
cp .env.example .env
```
Открой `.env` и впиши:
- `TELEGRAM_BOT_TOKEN` — токен от [@BotFather](https://t.me/BotFather) (`/newbot`)
- `LLM_API_KEY` — ключ Gemma (например, [Google AI Studio](https://aistudio.google.com/apikey))
- `EMBEDDINGS_API_KEY` — ключ для эмбеддингов (можно тот же Google-ключ)

> Остальное уже настроено по умолчанию под Google AI Studio + Gemma.
> Другой провайдер (OpenRouter/vLLM)? Поменяй `LLM_BASE_URL` и `LLM_MODEL`.

### 3. (Опционально) Добавить свои документы
Положи файлы (`.txt`, `.md`, `.pdf`, `.docx`) с инфой о грантах в `data/raw_docs/`.

### 4. Собрать базу знаний
```bash
python ingest.py                # скрейпит сайт фонда + твои файлы + строит индекс
# или только файлы, без скрейпинга:
python ingest.py --no-scrape
```

### 5. Запустить бота
```bash
python -m src.bot.main
```
Открой бота в Telegram, напиши `/start` и задай вопрос про гранты. 🎉

## ⚙️ Как это работает

1. **`ingest.py`** собирает тексты (сайт + твои файлы), режет на фрагменты,
   считает для них векторы (embeddings) и сохраняет индекс в `data/index/`.
2. **Бот** на каждый вопрос: превращает вопрос в вектор → находит ближайшие
   фрагменты → передаёт их Gemma как контекст → Gemma отвечает **только по этим
   данным** (без выдумок) и на языке вопроса.
3. Под ответом — ссылки на источники.

## 🔄 Обновление данных
Когда на сайте появились новые программы/новости — просто перезапусти:
```bash
python ingest.py
```
