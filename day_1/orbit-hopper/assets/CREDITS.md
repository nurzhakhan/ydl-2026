# Ассеты (все CC0 / Public Domain)

CC0 = можно использовать свободно, в т.ч. коммерчески, без указания авторства.
Атрибуция не требуется — список ниже для прозрачности и наших же будущих нужд.

## Графика
- `planets/planet*.png` — "20 CC0 Planet Sprites" (OpenGameArt, CC0)
  https://opengameart.org/content/20-cc0-planet-sprites — уменьшены до 256px.
- `img/bg.png`, `img/bg2.png` — Kenney "Space Shooter Redux" фоны (CC0)
  https://kenney.nl/assets/space-shooter-redux
- `img/meteor*.png` — Kenney "Space Shooter Redux" метеоры (CC0)

## Звук
