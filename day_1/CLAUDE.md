## What this is

YDL ("Y Data Lab") is a personal **learning project**: daily Python exercises, one per `day_N/` folder (e.g. `day_1/simple.py`). Each folder is self-contained. There are no dependencies, manifest files, or build system yet — they can be added as the work grows.

## Running code

- Run a file directly: `python day_N/<file>.py` (no virtualenv is set up).
- There is no test runner or linter configured yet.

## Working style

This is for learning, so explain the reasoning behind code and call out tradeoffs and alternatives rather than only handing over a finished answer.
